01 — Executive Summary

Project Overview

During Eid ul-Adha, millions of Bangladeshi families sacrifice animals. Finding a skilled koshai becomes a logistical nightmare. Koshai Finder bridges this gap with a modern two-sided marketplace. Users can book verified professionals, and koshais can manage their peak-demand schedules with ease.

02 — Problem & Solution

The Challenge & Our Answer

The Problem

  • Extreme demand spikes (10-50x) in 24h.
  • No centralized verification or booking.
  • Double bookings and frequent no-shows.
  • Zero price transparency or trust.

The Solution

  • Two-sided marketplace app.
  • Slot-based scheduling (no conflicts).
  • NID-verified professionals.
  • Real-time GPS tracking & chat.

Data Schema & Security

Collection: users

Collection: koshais
koshai_schema.json
{
  "isVerified": false,    // Admin verification
  "ratePerCow": 3500,     // BDT
  "status": "available",  // Live status
  "rating": 4.8,          // Avg stars
  "isEidMode": true       // Surge active
}

Firebase Security Rules ensure that users can only read their own data and koshais can only update their profile after verification.

06 — Core Logic

Kotlin Implementation

HomeViewModel.kt
@HiltViewModel
class HomeViewModel @Inject constructor(
  private val repo: KoshaiRepository
) : ViewModel() {

  private val _koshais = MutableStateFlow<List<Koshai>>(emptyList())
  val koshais: StateFlow<List<Koshai>> = _koshais

  fun loadKoshais(district: String) {
    viewModelScope.launch {
      repo.getAvailableKoshais(district).collect { list ->
        _koshais.value = list
      }
    }
  }
}
06.1 — Backend

Cloud Functions (Node.js)

reminders.ts
export const sendBookingReminders = functions.pubsub
  .schedule('every 30 minutes')
  .onRun(async () => {
    const now = admin.firestore.Timestamp.now();
    const snap24 = await db.collection('bookings')
      .where('status', 'in', ['confirmed'])
      .where('reminder24Sent', '==', false)
      .get();

    for (const doc of snap24.docs) {
      const b = doc.data();
      await sendFCM(b.userFcmToken, 'Reminder', 'Your koshai arrives tomorrow.');
      await doc.ref.update({ reminder24Sent: true });
    }
  });
